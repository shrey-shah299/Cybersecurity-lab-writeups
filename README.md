# 🛡️ Cybersecurity Lab Write-ups & Research

Welcome to my cybersecurity portfolio! This repository serves as a hands-on technical journal documenting my journey through advanced web application security labs, CTFs, and vulnerability research. 

The primary goal of this repository is to demonstrate a deep, mechanical understanding of web flaws, exploitation methodologies, and enterprise-grade remediation.

---

## Featured Platforms
* **PortSwigger Web Security Academy:** Focused on server-side and client-side web vulnerabilities.
* **Hacker101 CTF:** Focused on capture-the-flag methodologies and creative exploitation.

---

##  Repository Structure

The write-ups are organized by vulnerability category. Each lab folder contains a detailed `README.md` covering the vulnerability theory, discovery process, exploitation walkthrough (with screenshots), and remediation strategies.
